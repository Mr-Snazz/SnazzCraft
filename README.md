# SnazzCraft

--- All the dependencies are stated in dependancies.txt --- 


Building the Project:
  
  Simply go the the root directory and run makerscript.sh.
This should build the project and create a "worlds" folder
that is needed by the program. Everything I've used is cross
platform, so you can get this to compile on windows, though
I recommend using WSL (Windows Subsystem for Linux) for using
this on a Windows machine. Once build the executable will be 
in the bin folder (SnazzCraft/bin/ relative to the root directory)
and will be expected to be ran from within the SnazzCraft folder
which in and of itself is located within the root folder.


Controls:

WASD              - Movement
Space Bar         - Fly up
Shift             - Fly down
Q & E             - Rotate Yaw left and right respectively
X & C             - Rotate Pitch Down (X) and up (C)
Left Mouse Click  - Destroy voxel if able
Right Mouse Click - Place voxel if able (number keys determine the voxel to be placed, so try out a few)
